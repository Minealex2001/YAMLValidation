# YAMLValidation

YAMLValidation es una aplicación de escritorio multiplataforma desarrollada en Kotlin Compose para validar archivos YAML de forma sencilla y rápida.

## Funcionalidades principales
- Validación de archivos YAML.
- Interfaz intuitiva para cargar y analizar archivos YAML.
- Reporte de errores de sintaxis en los archivos.
- Consulta del historial de cambios (changelog) desde el pie de la ventana o desde la pantalla de configuración.

## Estructura del proyecto
* `/composeApp` contiene el código principal de la aplicación multiplataforma.
  - `commonMain`: código común para todos los targets.
  - `desktopMain`: código específico para la versión de escritorio.

## Recursos
- El archivo `changelog.md` está incluido como recurso y se puede consultar desde la aplicación.
